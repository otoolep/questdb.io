  • Docker

    Docker manifest to pull image for your target platform

    docker pull questdb/questdb

    Learn how to use it with our Docker guide.

  • Cross-platform binaries (requires JRE8)

    Supported Platforms: Windows 64-bit, Linux 64-bit, OSX and ARM64

    https://github.com/questdb/questdb/releases/download/5.0.0/questdb-5.0.0-bin.tar.gz (4Mb)

    Follow our installation steps to get started.

  • Homebrew

    Install with Homebrew

    brew install questdb

    Learn how to use QuestDB with Homebrew here.

  • Java library

    Embedded Java database as Maven dependency

    
<dependency>
    <groupId>org.questdb</groupId>
    <artifactId>core</artifactId>
    <version>5.0.0</version>
</dependency>

    Find out more in the embedded JAVA API documentation.
