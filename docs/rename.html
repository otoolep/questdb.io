RENAME TABLE

RENAME TABLE is used to change the name of a table.

Syntax

RENAME TABLE uses the following syntax

RENAME TABLE 'OLD_TABLE_NAME' TO 'NEW_TABLE_NAME';

Examples

RENAME TABLE 'test.csv' TO 'myTable';
